Sobre o projeto

Este projeto, em desenvolvimento, visa criar um sistema de gerenciamento para estações de recarga de veículos elétricos, utilizando o protocolo OCPP. O backend é construído em Node.js com Sequelize e MySQL, enquanto o frontend web usa Vue.js e o app móvel é desenvolvido em React Native com Expo.


Funcionalidades:
* Gerenciamento de Estações: Monitoramento do status e consumo de energia.
* App para Usuários: Localização de estações e pagamento de recargas.

O objetivo é oferecer uma solução integrada para administradores e usuários de estações de recarga.
